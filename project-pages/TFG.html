Design and development of a game concept with emphasis on the research and use of procedural generation techniques for the creation of solar systems.
This project investigates procedural generation techniques for creating solar systems and planets, culminating in a playable game prototype where players explore unique, procedurally generated environments.
Project Details / Background
Introduction: This project explores the design and development of a game that leverages procedural generation to create vast, explorable solar systems. Procedural generation techniques allow the creation of unique content dynamically, reducing manual labor while enhancing variability and scalability. The focus is on generating planets, solar systems, and explorable surfaces, ensuring each environment is distinctive and immersive.
Motivation: With the gaming industry moving towards larger, more detailed worlds, procedural generation has become essential for maintaining cost-effectiveness and innovation. This project draws inspiration from games like Minecraft, No Man’s Sky, and Astroneer, which utilize similar techniques to create dynamic, limitless environments. The aim is to contribute to this field by creating a game that combines solar system exploration and planetary surface adventures.
Technologies Used: The prototype employs advanced technologies and techniques, including:
  • Noise Maps: For generating planetary terrain and biomes.
  • Voxels: To simulate 3D landscapes, caves, and surfaces.
  • Wave Function Collapse: For creating coherent and non-repetitive structures.
  • ShaderGraphs: Used for rendering atmospheres, lighting, and planetary details.
Tools like Unity and Blender were essential for the implementation, while compute shaders were leveraged to maximize GPU efficiency for procedural calculations.
Core Features: The game offers two primary gameplay experiences:
  • Space Exploration: Navigate a procedurally generated solar system using an intuitive spaceship control system inspired by No Man’s Sky.
  • Planetary Exploration: Explore the surface of procedurally generated planets as an astronaut, complete with caves, biomes, and material analysis tools.
